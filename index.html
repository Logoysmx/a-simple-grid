Simple Grid

Description

It's a simple grid based in bootstrap and zurb foundation (this guide are using it).

Download

Download from github

Features

  • Use only a row container (width configurable, 1200 px by default)
  • Works with columns (by default are 12)
  • Custom prefix classes
  • If you need, can use more than 12 columns
  • Custom gutters (30px by default)
  • Without JavaScript
  • All browsers compatible

How to use

<div class="row">
<div class="col-6 in-cols">Content...</div>
<div class="col-6 in-cols">Content...</div>
</div >

It`s important use the class .in-cols when you need to use columns.

or...

<div class="row">
<div class="col-8 centered">Content...</div>
</div >
